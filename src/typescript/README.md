# TypeScript

A TypeScript playground examples and utilities.

## Overview

This directory contains TypeScript code samples, helper functions, and common patterns.

## Getting Started

```bash
# Install dependencies
npm install

# Run in development mode
npm run dev
```

## Structure

- `src/` - Source TypeScript files
- `dist/` - Compiled JavaScript output
- `types/` - Type definitions
